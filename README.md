# E-Commerce Sales EDA Dashboard

Exploratory Data Analysis and interactive dashboard built on the **Online Retail II** dataset (UCI Machine Learning Repository).

## Key Findings
- 📦 39,617 orders | 5,862 unique customers | £20M+ total revenue
- 🇬🇧 United Kingdom dominates revenue; EIRE and Netherlands lead internationally
- 📈 Clear seasonality: revenue peaks every November (holiday shopping)
- 🏆 Top product by revenue: REGENCY CAKESTAND 3 TIER
- 👥 22% VIP customers | 26% churned customers (retention opportunity)

## Tech Stack
- Python, Pandas, NumPy
- Plotly, Streamlit
- RFM Analysis, Customer Segmentation

## Project Structure
- `ecommerce_eda.ipynb` — Full EDA notebook
- `app.py` — Interactive Streamlit dashboard

## Dataset
[Online Retail II - UCI / Kaggle](https://www.kaggle.com/datasets/mashlyn/online-retail-ii-uci)
